# 8086 常用汇编指令速查表

| 类别                | 指令          | 缩写来源                                    | 功能说明                | 示例              | 备注                    |
| ------------------- | ------------- | ------------------------------------------- | ----------------------- | ----------------- | ----------------------- |
| **数据传输**        | `MOV`         | MOve                                        | 传送数据                | `MOV AX, BX`      | AX = BX                 |
|                     | `XCHG`        | eXCHanGe                                    | 交换寄存器内容          | `XCHG AX, BX`     | AX↔BX                   |
|                     | `LEA`         | Load Effective Address                      | 装载有效地址            | `LEA SI, [BX+DI]` | 不取内容，只取地址      |
|                     | `LAHF`        | Load AH from Flags                          | 将 FLAGS 低 8 位装入 AH | `LAHF`            | 便于判断标志            |
|                     | `SAHF`        | Store AH into Flags                         | AH → FLAGS 低 8 位      | `SAHF`            | 恢复标志                |
| **算术运算**        | `ADD`         | ADDition                                    | 加                      | `ADD AX, 1234h`   | AX += 1234h             |
|                     | `SUB`         | SUBtract                                    | 减                      | `SUB AX, BX`      | AX -= BX                |
|                     | `INC`         | INCrement                                   | 加 1                    | `INC CX`          | CX += 1                 |
|                     | `DEC`         | DECrement                                   | 减 1                    | `DEC DX`          | DX -= 1                 |
|                     | `MUL`         | MULtiply                                    | 无符号乘法              | `MUL BL`          | AX = AL\*BL             |
|                     | `IMUL`        | Integer MULtiply                            | 有符号乘法              | `IMUL BX`         | AX = AX\*BX             |
|                     | `DIV`         | DIVide                                      | 无符号除法              | `DIV BL`          | AX / BL → AL 余数 AH    |
|                     | `IDIV`        | Integer DIVide                              | 有符号除法              | `IDIV BX`         | AX / BX → AX 或 AL 余数 |
| **逻辑运算**        | `AND`         | AND                                         | 按位与                  | `AND AX, 0FFh`    | AX & 0FFh               |
|                     | `OR`          | OR                                          | 按位或                  | `OR AX, BX`       | AX = AX \| BX           |
|                     | `XOR`         | eXclusive OR                                | 按位异或                | `XOR AX, AX`      | AX 清零                 |
|                     | `NOT`         | NOT                                         | 按位取反                | `NOT AX`          | AX = ~AX                |
| **移位 / 循环**     | `SHL/SAL`     | Shift Left / Shift Arithmetic Left          | 左移（乘 2）            | `SHL AX, 1`       | AX <<= 1                |
|                     | `SHR`         | Shift Right                                 | 逻辑右移                | `SHR AX, 1`       | AX >>= 1，空位补 0      |
|                     | `SAR`         | Shift Arithmetic Right                      | 算术右移                | `SAR AX, 1`       | 保留符号位              |
|                     | `ROL`         | ROtate Left                                 | 循环左移                | `ROL AL, 1`       | 高位移到低位            |
|                     | `ROR`         | ROtate Right                                | 循环右移                | `ROR AL, 1`       | 低位移到高位            |
| **跳转 / 控制**     | `JMP`         | JuMP                                        | 无条件跳转              | `JMP label`       | 直接跳转                |
|                     | `JE/JZ`       | Jump if Equal / Jump if Zero                | ZF=1 跳转               | `JE equal`        | 相等跳转                |
|                     | `JNE/JNZ`     | Jump if Not Equal / Jump if Not Zero        | ZF=0 跳转               | `JNE notequal`    | 不等跳转                |
|                     | `JG/JNLE`     | Jump if Greater / Jump if Not Less or Equal | 有符号大于              | `JG label`        | SF=OF 且 ZF=0           |
|                     | `JL/JNGE`     | Jump if Less / Jump if Not Greater or Equal | 有符号小于              | `JL label`        | SF≠OF                   |
|                     | `CALL`        | CALL                                        | 调用子程序              | `CALL subroutine` | 压入返回地址            |
|                     | `RET`         | RETurn                                      | 返回                    | `RET`             | 弹出返回地址            |
|                     | `LOOP`        | LOOP                                        | 循环                    | `LOOP label`      | CX-1, CX≠0 跳转         |
| **堆栈操作**        | `PUSH`        | PUSH                                        | 压栈                    | `PUSH AX`         | SP-=2, 栈顶写入 AX      |
|                     | `POP`         | POP                                         | 弹栈                    | `POP BX`          | BX=栈顶, SP+=2          |
|                     | `PUSHA`       | PUSH All                                    | 压全部通用寄存器        | `PUSHA`           | AX,BX,CX,DX,SP,BP,SI,DI |
|                     | `POPA`        | POP All                                     | 弹出全部通用寄存器      | `POPA`            | 按序弹出                |
| **字符串操作**      | `MOVSB/MOVSW` | MOV String Byte/Word                        | 内存块拷贝              | `MOVSB`           | DS:SI → ES:DI           |
|                     | `LODSB/LODSW` | Load String Byte/Word                       | 从内存加载              | `LODSB`           | DS:SI → AL/AX           |
|                     | `STOSB/STOSW` | Store String Byte/Word                      | 写入内存                | `STOSB`           | AL/AX → ES:DI           |
|                     | `SCASB/SCASW` | SCAN String Byte/Word                       | 扫描字符串              | `SCASB`           | AL/AX ↔ ES:DI           |
|                     | `CMPSB/CMPSW` | CoMPare String Byte/Word                    | 比较字符串              | `CMPSB`           | DS:SI ↔ ES:DI           |
| **输入输出 I/O**    | `IN`          | INput                                       | 端口输入                | `IN AL, 60h`      | 从 60h 读               |
|                     | `OUT`         | OUTput                                      | 端口输出                | `OUT 60h, AL`     | 写 AL 到 60h            |
| **其他 / 标志操作** | `CLC`         | CLear Carry                                 | 清 CF                   | `CLC`             | 进位清零                |
|                     | `STC`         | SeT Carry                                   | 置 CF                   | `STC`             | 进位置位                |
|                     | `CLI`         | CLear Interrupt                             | 清中断允许              | `CLI`             | 禁止外部中断            |
|                     | `STI`         | SeT Interrupt                               | 置中断允许              | `STI`             | 允许外部中断            |
|                     | `NOP`         | No OPeration                                | 空操作                  | `NOP`             | 无操作，延时或占位      |
